# 通学路の安全ニュース 日次更新

全国で起きた通学路関連のリアルタイムニュースを取得・検証・記事化するワークフローを実行します。

## チーム構成

### 1. ニュースフェッチャー (news-fetcher)
全国の通学路関連ニュースをリアルタイムで収集:
- 通学路事故・事件のニュース検索
- 都道府県別のローカルニュース収集
- 警察・自治体の発表情報取得

### 2. ファクトチェッカー (fact-checker)
収集したニュースの正確性と信頼性を検証:
- ソースの信頼性評価
- 複数ソースによるクロスチェック
- 公式発表との照合

### 3. ニュースライター (news-writer)
検証済みニュースを読みやすい記事に編集:
- 客観的で正確な記事執筆
- 保護者・教育関係者向けの分かりやすい表現
- 重要ポイントの抽出

### 4. SEOオプティマイザー (seo-optimizer)
記事のSEOとメタデータを最適化:
- 検索キーワードの最適化
- メタタイトル・ディスクリプションの作成
- JSON-LD構造化データの生成

### 5. 画像ジェネレーター (image-generator)
ニュース記事用のサムネイルと図解を生成:
- ニュース内容に適したサムネイル画像生成
- 日本向けデザインの適用

## 実行方法

このコマンドを実行すると、5つのエージェントが順次連携して以下を行います:

1. **ニュース取得フェーズ**: WebSearchで最新ニュースを収集
2. **検証フェーズ**: 収集情報のファクトチェック
3. **執筆フェーズ**: 検証済み情報から記事を作成
4. **SEO最適化フェーズ**: メタデータと構造化データを設定
5. **画像生成フェーズ**: サムネイル画像を生成

## 出力先
- `lib/school-route-news.ts` - ニュースデータ
- `public/images/school-route-news/` - 画像ファイル

---

## ワークフロー開始

### Step 1: ニュース検索

以下のキーワードで最新情報を検索してください:

**交通事故検索キーワード:**
- "通学路 事故 速報 2026"
- "スクールゾーン 交通事故"
- "登下校 児童 事故"
- "横断歩道 小学生"

**不審者情報検索キーワード:**
- "通学路 不審者 2026"
- "児童 声かけ事案"
- "登下校 つきまとい"

**インフラ整備検索キーワード:**
- "通学路 ガードレール 設置"
- "スクールゾーン 整備"
- "歩道 拡幅 工事"

**施策検索キーワード:**
- "通学路 安全対策 自治体"
- "見守り活動 強化"

### Step 2: ファクトチェック

収集したニュースについて:
1. ソースの信頼性を確認（公式発表、大手メディア優先）
2. 複数ソースで内容をクロスチェック
3. 日時・場所・詳細の正確性を確認
4. 誤報・デマを除外

### Step 3: 記事作成

検証済みニュースから記事を作成:

```typescript
// lib/school-route-news.ts に追加するデータ形式
{
  id: "news-YYYY-MM-DD-NNN",
  slug: "prefecture-type-YYYYMMDD",
  title: "【都道府県名】タイトル",
  excerpt: "150文字以内の要約",
  content: "Markdown形式の本文",
  category: "accident" | "suspicious" | "infrastructure" | "policy" | "community",
  categoryLabel: "カテゴリー名",
  categoryColor: "#カラーコード",
  categoryIcon: "アイコン名",
  publishedDate: "ISO8601形式",
  location: {
    prefecture: "都道府県",
    city: "市区町村",
    area: "地域名"
  },
  tags: ["タグ1", "タグ2"],
  sources: ["ソース1", "ソース2"],
  keyPoints: ["ポイント1", "ポイント2", "ポイント3"],
  thumbnailUrl: "/images/school-route-news/thumbnails/slug.png",
  isBreaking: true/false,
  verifiedAt: "ISO8601形式"
}
```

### Step 4: 画像生成

サムネイル画像を生成するプロンプトガイドライン:

**交通事故の場合:**
- 事故現場を直接描かない（抽象的な表現）
- 注意喚起を促すデザイン
- 黄色い帽子、ランドセルなど日本の通学シーンの要素
- 警告色（赤、オレンジ）のアクセント

**不審者情報の場合:**
- 不審者を直接描かない
- 「注意」「警戒」を促すアイコン的表現
- 子どもの安全を守るイメージ
- オレンジ系の警戒色

**インフラ整備の場合:**
- ガードレール、歩道、信号などの安全設備
- 明るく前向きなデザイン
- 青系の安心感のある色

**地域活動の場合:**
- 見守りボランティアのイメージ
- 温かみのあるコミュニティ感
- 緑系の安心感のある色

### Step 5: データ更新

`lib/school-route-news.ts` の `NEWS_ITEMS` 配列に新しいニュースを追加してください。

古いニュース（90日以上前）は削除して、配列を最新の状態に保ちます。
