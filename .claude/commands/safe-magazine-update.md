# SAFE MAGAZINE コンテンツ更新

通学路安全ニュース・SAFEMAGAZINEのコンテンツを更新するチームワークフローを実行します。

## チーム構成

### 1. ニュースリサーチャー (news-researcher)
以下のカテゴリでリアルタイム情報を収集:
- 通学中の事故・事件ニュース
- 通学路見守りボランティア活動
- 子どもの防犯・安全教育
- 交差点・危険箇所の統計
- 自治体の安全施策

### 2. ファクトチェッカー (fact-checker)
- ソースの信頼性評価
- 複数ソースによるクロスチェック
- 統計データの正確性確認

### 3. 記事ライター (article-writer)
- ファクトチェック済み情報から記事作成
- 保護者・子ども向けコンテンツ
- SEO最適化

### 4. ビジュアルデザイナー (visual-designer)
- 記事サムネイル
- インフォグラフィック
- UIコンポーネント

### 5. 画像ジェネレーター (image-generator)
- Gemini 3 Pro Image Preview で日本向け高品質画像を生成
- サムネイル画像（16:9）
- 記事内インフォグラフィック（4:3）

### 6. SEOスペシャリスト (seo-specialist)
- メタタイトル・ディスクリプション最適化
- JSON-LD構造化データ生成
- Open Graph・Twitter Card設定

## 実行方法

このコマンドを実行すると、6つのエージェントが順次連携して以下を行います:

1. **リサーチフェーズ**: WebSearchとWebFetchで最新ニュースを収集
2. **検証フェーズ**: 収集情報のファクトチェック
3. **執筆フェーズ**: 検証済み情報から記事を作成
4. **デザインフェーズ**: 必要なビジュアル要素を提案
5. **画像生成フェーズ**: Gemini APIで日本向け画像を生成
6. **SEO最適化フェーズ**: メタデータと構造化データを設定

## 出力先
- `content/safe-magazine/research/` - リサーチ結果
- `content/safe-magazine/articles/` - 記事ファイル
- `content/safe-magazine/images/` - 画像仕様

---

## ワークフロー開始

$ARGUMENTS に指定されたカテゴリでリサーチを開始します。
カテゴリ未指定の場合は全カテゴリを対象とします。

利用可能なカテゴリ:
- `accident-news` - 事故ニュース
- `volunteer-activity` - 見守り活動
- `safety-tips` - 防犯・安全対策
- `danger-ranking` - 危険度ランキング
- `policy-update` - 施策・制度

### Step 1: ニュースリサーチ

以下のキーワードで最新情報を検索してください:

**事故ニュース検索キーワード:**
- "通学路 事故 2026"
- "スクールゾーン 交通事故"
- "登下校 児童 事故"

**見守り活動検索キーワード:**
- "通学路 見守り ボランティア"
- "スクールガード 活動"
- "地域 子ども見守り"

**防犯・安全対策検索キーワード:**
- "子ども 防犯 対策 2026"
- "不審者 対策 通学路"
- "子ども 安全教育"

**危険箇所検索キーワード:**
- "交差点 危険度 ランキング"
- "通学路 危険箇所 マップ"
- "事故多発 交差点"

**施策検索キーワード:**
- "通学路 安全対策 自治体"
- "スクールゾーン 規制強化"
- "通学路 安全プログラム"

リサーチ結果を `content/safe-magazine/research/` に保存し、次のエージェントに渡してください。

### Step 2: 画像生成

記事が完成したら、以下のコマンドで日本向け高品質画像を生成します:

```bash
npx tsx scripts/generate-safe-magazine-images.ts
```

**生成される画像:**

サムネイル画像:
- `public/images/safe-magazine/thumbnails/{slug}.png`

記事内画像:
- `public/images/safe-magazine/articles/{slug}/{image-id}.png`

**画像プロンプトガイドライン（日本向け）:**
- 黄色い帽子（通学帽）とランドセルを着用した小学生
- 日本の住宅街、横断歩道、グリーンベルト
- ガードレール、スクールゾーン標識
- 見守りボランティア、緑のおばさん
- アニメ/マンガ風の温かみのあるイラスト
- 教育コンテンツに適した親しみやすいデザイン

### Step 3: キャッシュクリアと確認

画像生成後、Next.jsのキャッシュをクリアして確認:

```bash
rm -rf .next/cache
npm run dev
```

ブラウザでハードリロード（Ctrl+Shift+R）して画像が正しく表示されることを確認。
