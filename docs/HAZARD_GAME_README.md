# 写真危険発見ゲーム - 実装完了！

## 🎯 概要

AIを活用した写真危険発見ゲームを実装しました。ユーザーが写真をアップロードすると、GPT-4o Visionが画像を分析して潜在的な危険要素を検出し、スコアとともに教育的なアドバイスを提供します。

## 🎮 ゲーム機能

### ゲームフロー
1. **写真アップロード** - カメラまたはファイル選択
2. **AI分析** (3-5秒) - GPT-4o Visionが危険要素を検出
3. **結果表示** - スコア、危険要素、安全アドバイス
4. **ポイント獲得** - ゲーミフィケーション連携

### スコアリングシステム
- 検出された危険数: 基本15点/個
- 高確信度ボーナス: +5点/個 (80%以上)
- 深刻な危険ボーナス: +10点/個 (レベル4以上)
- 最大スコア: 100点

### ランキングシステム
- **見習い分析者** (0-59点)
- **安全意識者** (60-69点)  
- **危険察知者** (70-79点)
- **安全分析者** (80-89点)
- **危険発見エキスパート** (90-94点)
- **安全マスター** (95-100点)

## 環境設定

### 必要な環境変数

`.env.local`ファイルに以下の環境変数を設定してください：

```bash
# OpenAI API設定（必須）
OPENAI_API_KEY=sk-your-openai-api-key-here

# オプション: OpenAI Organization ID
# OPENAI_ORG_ID=org-your-org-id-here
```

### OpenAI APIキーの取得方法

1. [OpenAI Platform](https://platform.openai.com/api-keys) にアクセス
2. アカウントを作成またはログイン
3. API Keysセクションで新しいAPIキーを作成
4. **重要**: `gpt-4o`モデルへのアクセス権限があることを確認してください

### トラブルシューティング

#### エラー: "画像の分析に失敗しました。画像形式やサイズを確認の上、もう一度お試しください。"

このエラーが発生する主な原因：

1. **OpenAI APIキーが設定されていない**
   - `.env.local`ファイルに`OPENAI_API_KEY`が設定されているか確認
   - APIキーが`sk-`で始まっているか確認

2. **gpt-4oモデルへのアクセス権限がない**
   - OpenAIアカウントでgpt-4oモデルが利用可能か確認
   - 必要に応じてOpenAIサポートに問い合わせ

3. **画像サイズが大きすぎる**
   - 最大20MBまでの画像をサポート
   - 大きい画像は事前にリサイズを推奨

4. **対応していない画像形式**
   - 対応形式: JPEG、PNG、GIF、WebP
   - その他の形式は事前に変換が必要

5. **レート制限**
   - OpenAI APIのレート制限に達している可能性
   - しばらく時間をおいてから再試行
