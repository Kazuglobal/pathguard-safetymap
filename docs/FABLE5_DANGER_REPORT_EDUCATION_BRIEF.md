# Fable5への指示書:「危険箇所レポート」機能の教育的改善分析

## 背景・目的

通学路安全マップアプリには、保護者・子ども向けに「危険箇所レポート」というPDF/PNG/JPEG出力機能がある。現状は登録済みルート周辺100m以内の危険箇所（交差点、標識のない横断歩道など）を検出し、件数・深刻度レベル別集計・個別の危険箇所カード（タイトル/カテゴリ/深刻度/説明/写真）を並べた「事務文書」的なレイアウトで出力している。

この機能を、単なる記録・共有ツールから**子どもと保護者が一緒に読んで学べる教育的なコンテンツ**へ進化させたい。分析と改善提案をFable5に依頼する。

## 現状の実装（分析対象）

- UI: `components/routes/route-danger-report-dialog.tsx`（プレビュー・写真選択・出力形式選択モーダル）
- データ: `lib/types.ts` の `DangerReport` / `RouteDangerReport` / `RouteDangerSummary`
  - 危険箇所ごとに `title`, `description`, `danger_type`（例: traffic）, `danger_level`（1〜4段階）, 写真URL, 緯度経度・住所を保持
- 出力ロジック: `lib/report-generation/route-danger-report.ts`
  - `jspdf` + `html2canvas` でHTML→canvas→PDF/PNG/JPEG変換
  - Mapbox Static Images APIで俯瞰地図＋危険レベル別色分けピンを生成
  - レポートHTMLは白背景・sans-serif・赤/オレンジ/黄の左ボーダー色分けのみで、装飾や教育的説明文は無い
- 参考: 同アプリ内の「きけんハンター」機能（`components/safety-quest/hunter/theme.tsx`）は「たんけんノート」という紙質感・丸ゴシック体（Zen Maru Gothic）・手帳/探検ノート風のデザイン言語を持ち、子ども向けの世界観が既に確立されている。

## Fable5への依頼事項

以下の観点でこの機能を分析し、改善案を出してほしい。

### 1. コンテンツ面の教育的改善
- 各危険箇所の「何が危険か」の説明を、子ども自身が読んで理解できる言葉（学年目安: 小学校低〜中学年）に言い換える具体例を出す
- 「どう行動すればよいか」（回避行動・注意ポイント）を各危険箇所カードに追加する場合の文言案
- 危険レベル（1〜4）を子どもにも直感的に伝える表現（色・アイコン・言い回し）の提案
- 保護者向けセクション（子どもとどう話し合うか、声かけの例）を追加する価値があるか

### 2. デザイン・トーン面の改善
- 既存の「きけんハンター/たんけんノート」の世界観（紙質感、丸ゴシック、探検ノート風レイアウト）とレポートのトーンを揃えるべきか、あえて事務文書寄りに留めるべきか、両方の長所短所
- 地図・写真・テキストのレイアウトを「学びのページ」として再構成する案（例: 見開き感、探検ログ風の演出）
- 印刷・PDF配布を前提としたときの可読性・印刷コスト（インク量）とのバランス

### 3. 機能面の拡張提案
- レポートを「読んで終わり」にせず、家庭内でのアクション（親子で現地確認するチェックリスト等）につなげる仕組み
- 学校・地域と共有する際に効果的な構成（教員・PTA向けサマリーページの要否）
- 既存のPDF/PNG/JPEG出力の枠組みを維持したまま実装可能な範囲か、大きな構造変更が必要か
