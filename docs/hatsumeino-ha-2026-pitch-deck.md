# ハツメイノハ 2026 企画書スライド構成

> **プロダクト名**: PathGuardian
> **キャッチコピー**: パパ・ママが主役。親子と地域の力で、通学路をもっと安全に
> **テーマ領域**: 防災 × AI × 地域コミュニティ

---

## ■ リポジトリ分析結果

### A. 技術スタック

| 領域 | 技術 |
|-----|------|
| フロントエンド | Next.js 16 + TypeScript + Tailwind CSS + Radix UI |
| バックエンド | マネージドDB（PostgreSQL + Auth + Storage + Realtime） |
| AI/ML | 画像解析AI（危険分析）、マルチモーダルAI（フォールバック） |
| 地図 | 地図API/SDK（ルーティング、ジオコーディング、等時間帯表示） |
| AR | WebAR / 3Dエンジン（リアルタイム危険ポイント表示） |
| 交通データ | 道路オープンデータ（日本の道路データプラットフォーム） |
| 3D | 3Dエンジン + WebGL連携 |
| デプロイ | クラウドホスティング |

### B. コア機能（5つ）

1. **AI危険検出・報告システム** — 親子で撮影した写真から画像解析AIが危険要素を自動分析、危険度レベル判定
2. **コミュニティ共有機能** — パパ・ママ友、地域住民とのヒヤリハット情報共有、いいね・コメント機能
3. **AR学習ナビ** — カメラ越しに登録済み危険ポイントをリアルタイム表示、親子で通学路を歩きながらARガイドで学ぶ
4. **危険探しミニゲーム** — 写真をアップロード → AIが危険箇所を検出 → ユーザーが危険箇所を当てるクイズ形式
5. **通学路危険箇所レポート・安全ニュース** — 登録ルート周辺100m以内（初期設定・調整可）の危険箇所一覧をPDF/画像でダウンロード、交通安全ニュース・季節別の安全ティップス配信

### C. 課題とターゲット

**社会課題**:
- 年間約10,000件の登下校中の交通事故（公的統計・出典要確認）
- **7歳（小学1年生）の交通事故率が最も高い**（公的統計・出典要確認）
- 紙の通学路マップは更新困難、リアルタイム共有不可
- **大人には見える危険が、子どもには見えない**
- 親子で安全を学ぶ機会が不足

**ターゲット**:
- **メイン**: 小中学生の子どもを持つ親子（特に小学校入学前後の家庭）
- **サブ**: 学校/PTA、自治体、地域住民（見守りボランティア）

### D. 柏の葉スマートシティ適合度

| テーマ | 適合内容 |
|-------|---------|
| 健康長寿 | 親子の安全意識向上＝次世代の健康な成長基盤 |

**活用可能な地域資産**: 地元大学キャンパス、地域デザインセンター、地域イノベーション拠点、街づくり事業者の実証フィールド、子育て世帯が多いニュータウン
