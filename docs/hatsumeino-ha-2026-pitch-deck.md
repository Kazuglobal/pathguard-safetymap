# ハツメイノハ 2026 企画書スライド構成

> **プロダクト名**: PathGuard SafetyMap
> **キャッチコピー**: AIと地域の力で、通学路をもっと安全に
> **テーマ領域**: 防災 × AI × 地域コミュニティ

---

## ■ リポジトリ分析結果

### A. 技術スタック

| 領域 | 技術 |
|-----|------|
| フロントエンド | Next.js 16 + TypeScript + Tailwind CSS + Radix UI |
| バックエンド | Supabase（PostgreSQL + Auth + Storage + Realtime） |
| AI/ML | **OpenAI GPT-4o Vision**（画像危険分析）、Google Gemini（フォールバック） |
| 地図 | **Mapbox GL**（ルーティング、ジオコーディング、等時間帯表示） |
| 交通データ | **xROAD API**（日本の道路データプラットフォーム） |
| 3D | Three.js + React Three Fiber |
| デプロイ | Vercel |

### B. コア機能（3つ）

1. **AI危険検出ゲーム** — GPT-4o Visionで写真から危険要素を自動分析、スコアリング、教育的アドバイス生成
2. **リアルタイム危険報告システム** — 位置情報・写真付き報告、自動ジオコーディング、危険度レベル判定
3. **ゲーミフィケーション** — ポイント・レベル・バッジ・ミッションで継続的エンゲージメント

### C. 課題とターゲット

**社会課題**:
- 年間約10,000件の登下校中の交通事故
- 紙の通学路マップは更新困難、リアルタイム共有不可
- 見守り活動が属人的で継続困難

**ターゲット**:
- 小中学生、保護者、学校/PTA、自治体、地域住民

### D. 柏の葉スマートシティ適合度

| テーマ | 適合内容 |
|-------|---------|
| 健康長寿 | 子どもの安全＝次世代の健康な成長基盤 |
| 新産業創造 | AI×地図×教育の新サービス創出 |
| 環境共生 | 徒歩通学の促進でCO2削減 |

**活用可能な地域資産**: 東京大学柏キャンパス、UDCK、KOIL、三井不動産実証フィールド

### E. 収益モデル

| モデル | 対象 | 内容 |
|-------|-----|------|
| B2C フリーミアム | 保護者 | 基本無料、プレミアム月額300円 |
| B2B SaaS | 学校/PTA | 年額12万円/校 |
| B2G | 自治体 | 年額50万円/自治体 |
