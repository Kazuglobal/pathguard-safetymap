# ハツメイノハ 2026 企画書スライド構成

> **プロダクト名**: PathGuard SafetyMap
> **キャッチコピー**: 親子と地域の力で、通学路をもっと安全に
> **テーマ領域**: 防災 × AI × 地域コミュニティ

---

## ■ リポジトリ分析結果

### A. 技術スタック

| 領域 | 技術 |
|-----|------|
| フロントエンド | Next.js 16 + TypeScript + Tailwind CSS + Radix UI |
| バックエンド | Supabase（PostgreSQL + Auth + Storage + Realtime） |
| AI/ML | **Google Gemini**（画像危険分析）、OpenAI GPT-4o Vision（フォールバック） |
| 地図 | **Mapbox GL**（ルーティング、ジオコーディング、等時間帯表示） |
| 交通データ | **xROAD API**（日本の道路データプラットフォーム） |
| 3D | Three.js + React Three Fiber |
| デプロイ | Vercel |

### B. コア機能（4つ）

1. **AI危険検出・報告システム** — 親子で撮影した写真からGeminiが危険要素を自動分析、危険度レベル判定
2. **コミュニティ共有機能** — パパ・ママ友、地域住民とのヒヤリハット情報共有、いいね・コメント機能
3. **通学路危険箇所レポート** — 登録ルート周辺100m以内の危険箇所一覧をPDF/画像でダウンロード、事前学習に活用
4. **安全ニュース・ティップス** — 交通安全ニュース、防犯教育コンテンツ、季節別の安全ティップスを配信

### C. 課題とターゲット

**社会課題**:
- 年間約10,000件の登下校中の交通事故
- 紙の通学路マップは更新困難、リアルタイム共有不可
- 見守り活動が属人的で継続困難
- 親子で安全を学ぶ機会が不足

**ターゲット**:
- **メイン**: 小中学生の子どもを持つ親子
- **サブ**: 学校/PTA、自治体、地域住民（見守りボランティア）

### D. 柏の葉スマートシティ適合度

| テーマ | 適合内容 |
|-------|---------|
| 健康長寿 | 親子の安全意識向上＝次世代の健康な成長基盤 |
| 新産業創造 | AI×地図×コミュニティの新サービス創出 |
| 環境共生 | 徒歩通学の促進でCO2削減 |

**活用可能な地域資産**: 東京大学柏キャンパス、UDCK、KOIL、三井不動産実証フィールド、子育て世帯が多いニュータウン

### E. 収益モデル（API費用を考慮）

#### API費用試算（月間）

| 規模 | Mapbox費用 | Gemini費用 | 合計API費用 |
|-----|-----------|-----------|------------|
| 1,000ユーザー | 無料枠内 | 約500円 | 約500円/月 |
| 10,000ユーザー | 約30万円 | 約5,000円 | 約31万円/月 |
| 100,000ユーザー | 約200万円 | 約5万円 | 約205万円/月 |

※ Mapbox: 無料枠（50,000 map loads/月）超過分 $5/1,000 loads
※ Gemini: 画像分析 約$0.001/画像（Flash利用時）

#### 収益源とプライシング

| モデル | 対象 | 内容 | 根拠 |
|-------|-----|------|-----|
| B2C フリーミアム | 親子 | 基本無料、プレミアム月額**500円** | API費用+利益を確保 |
| B2B SaaS | 学校/PTA | 年額**18万円/校** | 管理機能+レポート生成 |
| B2G | 自治体 | 年額**80万円/自治体** | データ分析+政策提言 |
